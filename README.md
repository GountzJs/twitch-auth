# Twitch Auth

## Getting Started

### Prerequisites

- Node.js
- Pnpm
- Vite
- Tailwind CSS
- React
- Typescript

### Installation

1. Clone the repository

2. Install dependencies

```bash
pnpm install
```

3. Run the development server

```bash
pnpm dev
```

4. Open your browser and navigate to `http://localhost:3000`
